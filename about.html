About BlockVerse

      BlockVerse is a simple gaming website where users can explore popular and trending games in a clean and organized way.
      
      The website includes a homepage with a featured section, a games page that shows different game cards,
      
      a gallery with screenshots, and a contact page for users to reach out.
      
      BlockVerse is designed to be easy to navigate, visually appealing, and mobile-friendly so that users
      
      can browse smoothly on any device. The site includes features like a neon-themed design, organized card layouts,
      
      a responsive grid gallery, and clear navigation links on every page. 

      Each game card displays an image, title, and a short description so users can quickly understand what the game is about. 

      The gallery page uses a dynamic grid that shows different game images in unique layouts.

      The contact page has a simple form where anyone can send a message or share feedback.

      BlockVerse focuses on showcasing games in a clean and user-friendly format while
      
      keeping the design modern and glowing with a gaming theme.
